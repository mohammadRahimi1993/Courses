# 📊 Big-Data-Fundamentals-Practice-2022

<p align="justify">
Welcome to the <b>Big Data Fundamentals & Practice</b> repository. This collection was prepared in 2022 by <b>Mohammad Rahimi</b>, a PhD candidate at the University of Isfahan, Iran. It is designed to introduce students and beginners to the fundamental concepts of Big Data, the Hadoop ecosystem, Apache Spark, and practical MapReduce programming. The tutorials combine theoretical explanations with hands-on demonstrations using Google Colab, providing an accessible environment for learning Big Data technologies without requiring a dedicated cluster. To view the complete training courses, please visit https://programmingschool.ir.
</p>

## 📚 Big Data Tutorials

**Part 1: Introduction to the Hadoop Ecosystem**
This tutorial introduces the Hadoop ecosystem and demonstrates how to set up and run a Hadoop environment using Google Colab, providing a simple platform for experimenting with distributed data processing.

**Part 2: Introduction to Apache Spark**
This tutorial presents the Apache Spark engine and explains its architecture, advantages, and performance improvements compared with the traditional Hadoop MapReduce framework.

https://www.aparat.com/v/ext3eu6

**Part 3: MapReduce Programming Practice**
In this hands-on tutorial, a practical MapReduce example is implemented and solved in the Google Colab environment, illustrating the Map and Reduce phases through a complete programming exercise.

https://www.aparat.com/v/pxc19na
